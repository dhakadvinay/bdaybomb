STEP 1: CREATE DATABASE
================================================================================
1. Open phpMyAdmin: http://localhost/phpmyadmin
2. Click "New" in left sidebar
3. Enter database name: birthday_surprise
4. Click "Create"
5. Select the new database
6. Go to "SQL" tab
7. Copy-paste this SQL and click "Go":

---BEGIN SQL---
CREATE TABLE IF NOT EXISTS admin_users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    password VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE IF NOT EXISTS photos (
    id INT PRIMARY KEY AUTO_INCREMENT,
    filename VARCHAR(255) NOT NULL,
    uploaded_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

STEP 3: CREATE UPLOADS FOLDER
================================================================================
The /uploads/ folder will be created automatically on first photo upload.
If you want to create it manually:
1. In C:\xampp\htdocs\myfirstweb\, create a folder named "uploads"
2. Right-click > Properties > Security > Edit
3. Select "Users" and check "Full Control"
4. Click OK

TROUBLESHOOTING
================================================================================

‚ùå "Connection failed" error
   ‚Üí Check XAMPP MySQL is running (green indicator)
   ‚Üí Verify database "birthday_surprise" exists
   ‚Üí Check includes/db.php has correct credentials

‚ùå Photos not uploading
   ‚Üí Check /uploads/ folder exists and is writable
   ‚Üí Verify file size is under 5MB
   ‚Üí Ensure file is JPG or PNG

‚ùå Admin login not working
   ‚Üí Clear browser cookies (Ctrl+Shift+Delete)
   ‚Üí Verify admin_users table has data
   ‚Üí Try default: admin / admin123

‚ùå Animations not working
   ‚Üí Check internet connection (GSAP/Lottie from CDN)
   ‚Üí Open browser console (F12) for errors
   ‚Üí Clear browser cache (Ctrl+F5)
